从首都机场打车回学校的路上，望着窗外的树，两年前我乘坐大兴机场线从大兴进城的记忆突然变得鲜活。记得当时，车窗外也是这样的树……

我的视力不太好，以为树上还有叶子，但是，时值北京的二月，正是天寒地冻的季节，长得高高的树干上哪怕只剩一片叶子也会被吹落。树梢那些细枝条此时成为了树的叶子，而其上面较粗的那些枝桠，便成了托起叶子的枝干。所以，北京的树一到冬天，便长出许多雾来。黄的灰的雾团在树上安静地生长，即便再怎么天寒地冻风吹雨打也不散去。即使是有叶子的树——这种季节也只剩下松树了，墨绿色的松针从远处看好像被一片大火烧过似的漆黑，从那当中辨别不出什么别的颜色。就算你心怀期待地在视野里捕获了一株黄色的树，仔细一看也发现那不过是些细软的树枝。这一带的树木向来擅长伪装自己。

邱老师寒假来广州玩，下飞机后跟我说的第一句话便是：“广州好绿”。是这样的。坐在和谐号上驶向广州时，视野所及都是绿色的树。高大的绿色树木一簇一簇挨在一起，从车窗往外看，远处山头上本该高大的杉树好像一丛苔藓挤在小山丘上。我时常想象是否有一双巨大的手来抚摸大地身上的绒毛。但北京是不一样的……这里的冬天赤裸着皮肤接受寒风拷问，只有灰蒙蒙的雾笼罩在地表上，永远不会散去。

夕阳往下落了，我逐渐明白那片雾是什么。太阳没法照透的阴霾随着春天的到来会变好吗？到时候，我又能在北京看到真正的树了。车继续往前开，而我因为一天的奔波在车上沉沉睡去。

<br>

不知为何我的写作风格总是会模仿最近阅读的作家。在家收拾行李时，书架上读了一小半的阿列克谢耶维奇的《锌皮娃娃兵》好像在呼唤我，让我路上把它带着。于是，我把它塞进了本就沉重的包里。于是，我在华南丘陵和华北平原的上空一直读这本书，那天晚上，在洗被子的时候，一口气将它全读完了。

很难说这是一本多么有趣的书，但它就是有让人继续看的力量。或者说是拥有令人**害怕不去读完它**的力量。《锌皮娃娃兵》讲述的是苏阿战争的故事，阿列克谢耶维奇将战争幸存者的自述与烈士家属们的只言片语编织成了一部纪实性的作品。战争极大地改变了当时的苏联人民，使得那些原本热爱祖国、由于无知投入战争的家庭支离破碎，许多人只能在战场上寻求生存的真实。这也就是为什么有许多退役老兵要求回到阿富汗——一旦参与过战争，杀了人，就再也不可能回到曾经普通的生活中去了……无辜的民众一旦成了政治错误的牺牲品，代价是巨大的。简单概括书的内容那就是：一些小小的和平主义震撼。然而，和平主义应该成为谴责士兵们的理由吗？他们只是政治错误的牺牲品。许多人去前线是抱着“到祖国需要我的地方去”的信念。一位女儿对她母亲说的话让我印象深刻：

{% note danger no-icon %}
“反正需要有人去，我想到困难的地方去。”
{% endnote %}

那么就用这位母亲的话结束这篇随笔吧。

{% note info no-icon %}
我坐在小坟旁……一个个做母亲的，带着孩子从面前经过……我听到她们在说：“这算什么母亲呀？在咱们这个时代，她怎么能把自己的独生女儿送上战场？把一个女孩子交出去？”

我女儿的墓碑上刻着“纪念独生女儿”。

她们竟敢这么说，竟敢这样讲！她是宣过誓的，她是护士，外科医生吻过她的手。她去那边，是为了救死扶伤，为了拯救她们的儿子……
{% endnote %}
