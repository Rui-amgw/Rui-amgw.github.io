# hash #
标题。hash个数=标题等级

*star*
斜体

**double star**
粗体

{% note danger no-icon %}
红色引用框
{% endnote %}

{% note %}
灰色引用框
{% endnote %}

{% note info no-icon %}
蓝色引用框
{% endnote %}

hexo内置的tag参见[这篇文章](https://hexo.io/docs/tag-plugins)

引用：
{% blockquote [author[, source]] [link] [source_link_title] %}
content
{% endblockquote %}

代码：
{% codeblock [title] [lang:language] [url] [link text] [additional options] %}
code snippet
{% endcodeblock %}
